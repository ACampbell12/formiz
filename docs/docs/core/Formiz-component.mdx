## Props

#### `onSubmit(values, actions)`

> Triggered when the form is **submitted**.

#### `onValidSubmit(values, actions)`

> Triggered when the form is **valid** and **submitted**.

#### `onInvalidSubmit(values, actions)`

> Triggered when the form is **invalid** and **submitted**.

#### `onChange(values, actions)`

> Triggered every time the form change.

> ⚠️ This is triggered when each field is mounted.
> ℹ️ You can get `values` of the form with the useForm() hook.

#### `onValid()`

> Triggered when the form is **valid**.

> ℹ️ You can get `isValid` value with the useForm() hook.

#### `onInvalid()`

> Triggered when the form is **invalid**.

> ℹ️ You can get `isValid` value with the useForm() hook.

#### `autoForm`

> Set to `true` to auto add a `<form>` element with auto onSubmit.

> ℹ️ You can get the `submit()` method with the useForm() hook.
